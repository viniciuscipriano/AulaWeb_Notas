# Registrador de Alunos

Este projeto consiste em um registrador simples de alunos em uma tabela HTML, usando HTML, CSS e JavaScript.

## Estrutura do Projeto

O projeto é composto por três arquivos:

- **index.html**: Define a estrutura da página do registrador de alunos.
- **style.css**: Arquivo CSS que contém estilos para personalizar a aparência da página.
- **main.js**: Arquivo JavaScript que controla o comportamento dinâmico da página, incluindo a adição de alunos à tabela.

## Funcionamento

- A página apresenta um formulário onde o usuário pode digitar o nome e a nota de um aluno.
- Quando o usuário envia o formulário, a função `addTabela()` é acionada, que adiciona os dados do aluno à tabela abaixo do formulário.
- Os dados inseridos são exibidos em uma tabela HTML com duas colunas: "NOME" e "NOTA".
- Após a adição do aluno à tabela, os campos do formulário são limpos para permitir a inserção de novos dados.

## Observações

- Este é um projeto básico para demonstrar como adicionar dados a uma tabela HTML dinamicamente.
- Não há validação de dados neste projeto. Em uma aplicação real, é recomendável adicionar validação de entrada de dados para garantir a precisão e a integridade dos dados inseridos.
